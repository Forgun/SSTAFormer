 SSTAFormer
Code for the manuscript: A Spatial Multi-Scale and Adaptive Filtering Attention Framework for Industrial Soft Sensor

## 1. Code Description
This repository contains the core code of the SSTAFormer model, including model construction, training and testing scripts.

## 2. Dataset
The datasets used in this study are public benchmark datasets for industrial soft sensing (Sulfur Recovery Unit and Debutanizer Column), which are publicly available from open process control data repositories.

## 3. Reproducibility
All experimental results reported in the manuscript can be fully reproduced using the provided code and public datasets.
